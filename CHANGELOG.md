# Changelog

All notable changes to lifeOS will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] — 2025-03-15

### Added

- **Today Dashboard** — Daily command center with tasks, habits, journal, and quick capture
- **Universal Inbox** — Capture anything with minimal friction, triage later
- **Tasks** — Full task management with priorities, due dates, projects, and tags
- **Habits** — Daily habit tracking with streaks, grace days, and domain grouping
- **Journal** — Markdown journal entries with mood, energy, and tag tracking
- **Notes** — Markdown-first knowledge base with tags and full-text search
- **Ideas** — Capture and develop ideas through stages (seed → validated)
- **People** — Relationship tracking with contact details and interaction logging
- **Learning** — Track books, articles, courses, and other learning resources
- **Projects** — Active project management with status tracking
- **Goals** — Long-term goal setting with milestones and progress tracking
- **Health** — Log sleep, mood, energy, symptoms, and workouts
- **Finance** — Simple expense and income tracking
- **Weekly Reviews** — Auto-generated reviews from real data across all domains
- **Graph Explorer** — Visual connections between entities
- **Timeline** — Chronological view across all activity
- **Insights** — Analytics and patterns across your data
- **Gamification** — XP system, streaks, and domain levels
- **Full-Text Search** — SQLite FTS5 across all content
- **Export & Backup** — JSON export of all data from Settings
- **Passphrase Auth** — bcrypt-based single-user authentication
- **Docker Support** — Production-ready Dockerfile and docker-compose.yml
- **Seed Data** — Optional demo data for exploring the app
